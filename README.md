# Smart Agro v1

1. Clona projeto

requisito:
- Git

```bash
git clone https://github.com/douglasamorim13/smartagro-app.git
```

2. Instalar dependências no projeto frontend

requisito:
- Node.js V20.X

```bash
cd frontend
npm install
```

Rodar frontend

```bash
npm run dev
```

3. Instalar dependências no projeto backend

requisito:
- Node.js V20.X

```bash
cd backend
npm install
```

Rodar backend

```bash
npm start
```

Projeto rodando localmente em: http://localhost:5173
